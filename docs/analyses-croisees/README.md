# Analyses croisées

Cette arborescence sert à **rédiger les analyses croisées** entre les trois familles de frameworks (spec-kit, BMAD, OpenSpec). Elle s’appuie sur les analyses spécifiques de [../analyses-specifiques/](../analyses-specifiques/).

## Structure

- **documents/** — Documents de travail, brouillons et supports pour les comparaisons
- **par-thematique/** — Analyses croisées organisées par thème (concepts, processus, artefacts, etc.)
- **par-paire/** — Comparaisons deux à deux :
  - spec-kit vs BMAD
  - spec-kit vs OpenSpec
  - BMAD vs OpenSpec
- **synthese/** — Synthèse globale des analyses croisées et conclusions transverses

## Utilisation

1. Remplir d’abord les analyses spécifiques (documents, référentiels, synthèses par famille).
2. Produire les comparaisons dans **par-thematique** ou **par-paire** selon l’angle choisi.
3. Consolider les résultats dans **synthese**.
