# Analyses croisées par thématique

Comparaisons entre les trois familles organisées **par thème** (concepts, processus, artefacts, gouvernance, etc.).

Exemples de thématiques possibles :

- Processus et workflows
- Artefacts et livrables
- Rôles et responsabilités
- Intégration outillage / IA
