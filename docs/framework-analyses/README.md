# Framework-Specific Analyses

This directory contains **dedicated analyses for each framework family**
(spec-kit, BMAD, OpenSpec). Each subdirectory includes:

- **documents** — source documents, excerpts, reading notes
- **glossaries** — glossaries, taxonomies, approach-specific models
- **descriptions** — detailed descriptions of concepts, processes and artifacts
- **summary** — summaries, abstracts and conclusions per framework

## Frameworks

| Framework | Directory                | Description                  |
|-----------|--------------------------|------------------------------|
| spec-kit  | [spec-kit/](./spec-kit/) | GitHub spec-kit approach     |
| BMAD      | [bmad/](./bmad/)         | BMAD method and modules      |
| OpenSpec  | [openspec/](./openspec/) | Fission-AI OpenSpec approach |
