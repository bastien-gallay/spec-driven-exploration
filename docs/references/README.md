# Reference Repositories

Read-only cloned repositories for AI exploration, referencing and partial reuse.

## Structure

- **spec-kit** — [github/spec-kit](https://github.com/github/spec-kit)
- **BMAD** — BMAD method and modules (4 repos)
- **OpenSpec** — [Fission-AI/OpenSpec](https://github.com/Fission-AI/OpenSpec)

## Installing References

Run the following commands from the project root:

```bash
# spec-kit
git clone git@github.com:github/spec-kit.git docs/references/spec-kit

# BMAD
git clone git@github.com:bmad-code-org/BMAD-METHOD.git \
  docs/references/BMAD/BMAD-METHOD
git clone git@github.com:bmad-code-org/bmad-module-game-dev-studio.git \
  docs/references/BMAD/bmad-module-game-dev-studio
git clone \
  git@github.com:bmad-code-org/bmad-module-creative-intelligence-suite.git \
  docs/references/BMAD/bmad-module-creative-intelligence-suite
git clone git@github.com:bmad-code-org/bmad-method-wds-expansion.git \
  docs/references/BMAD/bmad-method-wds-expansion

# OpenSpec
git clone git@github.com:Fission-AI/OpenSpec.git docs/references/OpenSpec
```

## Updating

In each cloned subdirectory: `git pull`.
