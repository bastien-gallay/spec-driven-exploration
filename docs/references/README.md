# Reference repositories

Repositories clonés en lecture seule pour exploration IA, référencement et réutilisation de parties.

## Structure

- **spec-kit** — [github/spec-kit](https://github.com/github/spec-kit)
- **BMAD** — Méthode et modules BMAD (4 repos)
- **OpenSpec** — [Fission-AI/OpenSpec](https://github.com/Fission-AI/OpenSpec)

## Installation des références

Exécuter les commandes suivantes depuis la racine du projet :

```bash
# spec-kit
git clone git@github.com:github/spec-kit.git docs/references/spec-kit

# BMAD
git clone git@github.com:bmad-code-org/BMAD-METHOD.git docs/references/BMAD/BMAD-METHOD
git clone git@github.com:bmad-code-org/bmad-module-game-dev-studio.git docs/references/BMAD/bmad-module-game-dev-studio
git clone git@github.com:bmad-code-org/bmad-module-creative-intelligence-suite.git docs/references/BMAD/bmad-module-creative-intelligence-suite
git clone git@github.com:bmad-code-org/bmad-method-wds-expansion.git docs/references/BMAD/bmad-method-wds-expansion

# OpenSpec
git clone git@github.com:Fission-AI/OpenSpec.git docs/references/OpenSpec
```
