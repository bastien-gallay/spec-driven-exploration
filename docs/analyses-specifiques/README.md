# Analyses spécifiques par famille de framework

Cette arborescence regroupe les **analyses dédiées à chaque famille** de framework (spec-kit, BMAD, OpenSpec). Chaque sous-dossier contient :

- **documents** — documents sources, extraits, notes de lecture
- **referentiels** — glossaires, taxonomies, modèles propres à l’approche
- **descriptions** — descriptions détaillées des concepts, processus et artefacts
- **synthese** — synthèses, résumés et conclusions par famille
